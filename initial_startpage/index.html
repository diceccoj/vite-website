Initial

This is an ongoing project I call Initial. A browser startpage you can use right now!
Like this website, Initial uses HTML, Javascript and CSS. While this website mainly uses CSS to control styling, my goal with this website was to explore Javascript more, which allowed for many useful features to be added to the website that CSS couldn't do.

So far this project features:
- A dark and light theme
- Two widget alignments: center and left
- A weather widget using OpenWeatherMap API
- A background theme to your choosing using royalty-free photos from UnsplashAPI

Billing System

This is a billing system for "Jonathan's Cafe" created in C. It takes the inputs of every item in the store bought, calculates the amount of money spent on each products, including the money spent on total beverages and food products, then displays it to the user. Then, a bill is printed out for the user, unless the user did not buy anything. In that case, the program ends there.
